Packages is nothing but Collection of Files or classses in a single Folder.
There are 4 types of access Modifiers

1. Public
2. Default
3. Private
4. Protected.

All the Members of the class which are variables, methods and inner classes are accessed within the class for all the access modifiers.

Outer Class can only be default and public.

A class can be used as by creating an object of a class(hasA) also called as Non Sub Class and By inheriting (isA) Also called as Sub Class
